# Project 01: M365 Tenant Setup Simulation

**Student:** Katlego Kingsley Vilakazi  
**Platform:** Azure VM (Windows Server 2022)  
**Simulation of:** Microsoft 365 Tenant using Active Directory

## Overview
This project simulates the setup of a Microsoft 365 tenant using an Azure virtual machine with Windows Server 2022. Active Directory was used to create Organizational Units (OUs), users, folders, and permissions to reflect a real-world M365 environment.

## Project Tasks Completed
- Created Organizational Units:
  - Employees, HR, IT, Management, Research, Stakeholders, Accounting, Auditors
- Created Users:
  - Sarah, Ted, Phil, Murtaugh, Spencer, Tim, Dean, Jeanine, Pearl, Frank, Sally, Bruce, Stephen, Guy
- Created folders and subfolders in `PromisedLand`:
  - Employees, HR, IT, Management, Research, Accounting, Stakeholders, Auditors
- Assigned NTFS permissions according to the project requirements
- Assigned users to the appropriate OUs
