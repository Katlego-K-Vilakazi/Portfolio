# User Management API

A REST-style API built with Flask and SQLite that allows retrieval and modification of user authorization levels.

## Overview

This project simulates credential and authorization management in a secure system.  
It allows clients to retrieve user records and update authorization levels via URL parameters.

This project demonstrates backend API development, database integration, routing logic, and automated testing.

---

## Technologies Used

- Python 3
- Flask
- SQLite3
- Pytest

---

## Features

- Retrieve a user by username
- Update a user’s authorization level
- JSON responses
- Error handling for invalid requests
- Automated test validation
